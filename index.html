Introduction

In 0.13.0, Tommaso introduced Data-Based Modding. This type of modding happens in the textures and will be compatiple between Android and iOS

To install this without blocklauncher, you have to patch the texture yourself, which can be tedious.
Note: Do NOT hand out anything you get from the com.mojang.minecraft.apk except the assets folder. Do NOT hand out the modified .apk.


A video tutorial can be found Here

Normal Item

This is a basic item


	{
		"name": "customItem",
		"id": 254, // 256 + 254 = 510
		"icon": "stick", // image of item
		"category": "Miscellaneous", // creative tab thingy
		"foil": true, // show enchant glint
		"stacked_by_data": true, // 
		"hand_equipped": false, // does it look like a tool in hand
		"hover_text_color": "aqua", // if not defined, will be white
		"max_stack_size": 1 // 
	},

Food Item

This is a basic food item


	{
		"name": "customItem",
		"id": 254, // 256 + 254 = 510
		"icon": "stick", // image of item
		"category": "Miscellaneous", // creative tab thingy
		"use_animation": "eat"
		"use_duration": 32,
		
		"food": {
			"nutrition": 6,
			"saturation_modifier": "normal",
			"using_converts_to": "bowl",
			"is_meat": false,
			"eat_sound": "random.burp",
			"effects": [
				{
					"name": "poison",
					"chance": 1.0,
					"duration": 5,
					"amplifier": 0
				},
				{
					"name": "nausea",
					"chance": 1.0,
					"duration": 5,
					"amplifier": 0
				}
			]
		}
	},

The "saturation_modifier" tag can be one of:

  • poor
  • low
  • normal
  • good
  • supernatrual

The "using_converts_to" tag converts the item to the said item after eating it
The "is_meat" tag is used for stuff like wolf and such, I believe.
The "eat_sound" tag defines which sound to play, find them in the assets
The "effects" tag is an array which defines which effects are used when you eat it.
The "use_animation" tag defines which animation to use, can be one of:
  • none
  • eat
  • drink
  • block
  • bow

Seed Item

This is a basic seed item


	{
		"name": "customItem",
		"id": 254, // 256 + 254 = 510
		"icon": "stick", // image of item
		"category": "Miscellaneous", // creative tab thingy
		
		"seed": {
			"crop_result" : "carrots",
			"plant_at": "farmland"
		}
	},

The "seed" tag can be used a setTile alternative, which can look like this


		"seed": {
			"crop_result" : "stone",
			"plant_at": "grass"
		}
